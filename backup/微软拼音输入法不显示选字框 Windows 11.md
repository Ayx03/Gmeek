重启 Windows 资源管理器（explorer.exe）即可。可以使用任务管理器的详细信息选项卡（当然在进程选项卡找也可以），也可以在 PowerShell 中执行命令：`taskkill /IM explorer.exe /F; explorer`，如果结束了资源管理器但是没有重启可以使用 Ctrl + Alt + Del 组合键打开任务管理器。
